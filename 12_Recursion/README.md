# 12. Recursion

A function that calls itself to solve smaller instances of the same problem.
Every recursive function needs a base case and a recursive case.

## Files

| File | Description |
|------|--------------|
| `01_factorial.cpp` | Factorial using recursion |
| `02_fibonacci.cpp` | Fibonacci sequence using recursion |
| `03_gcd_euclidean.cpp` | GCD via the Euclidean algorithm |
| `04_tower_of_hanoi.cpp` | The classic Tower of Hanoi puzzle |

## Key Takeaways
- Same principles as in C — the difference here is purely syntactic (C++ I/O, references, etc.).
- Recursive solutions are common in tree/graph traversal and divide-and-conquer algorithms.
